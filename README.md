資管所企業資料通訊作業 - 簡易CLI聊天室 TCP Client/Server

鄭新霖 aka Hsin-lin Cheng <<lancetw@gmail.com>>

- - - - 

Command:

    $ make

Optional:
	
    $ make server && make client

DEBUG mode:

    $ make debug

Use (server side):

    $ ./server
    
Use (client side):

    $ ./client 127.0.0.1
